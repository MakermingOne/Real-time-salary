# 实时工资 - 产品需求文档

## 1. 产品概述

**产品名称：** 实时工资  
**产品类型：** PWA 移动应用 / 网页应用  
**核心功能：** 实时显示用户每分每秒的工资收入，支持多种计算方式和主题风格  
**目标用户：** 通过工资收入可视化激励自己的职场人士

---

## 2. 功能需求

### 2.1 基础设置

| 功能 | 说明 |
|------|------|
| 月薪输入 | 支持输入工资金额 |
| 货币单位 | 支持人民币(¥)和美元($) |
| 时区选择 | 支持北京、伦敦、纽约、东京、洛杉矶时区 |

### 2.2 收入计算方式

| 模式 | 说明 | 计算规则 |
|------|------|----------|
| 24小时计算收益 | 从当天0点开始，按每秒平均计算 | 月薪 ÷ 当月自然天数 ÷ 86400秒 |
| 仅工作时间计算收益 | 按实际工作日和工时计算 | 仅在工作时间内增长，午休不计算 |

**计算方式说明：**
- 24小时模式：每天从0点开始，任何时间打开都能看到收入增长
- 工作时间模式：只在设置的上班时间内增长，周末不计算，午休时间不计入

### 2.3 工作时间设置（仅工作时间模式）

| 设置项 | 默认值 |
|--------|--------|
| 上班时间 | 09:30 |
| 下班时间 | 19:00 |
| 午休开始 | 12:30 |
| 午休结束 | 14:00 |

### 2.4 视觉风格

**主题风格：**
- 暗黑模式（默认）
- 亮色模式
- 金色模式
- 绿色模式
- 紫色模式

**背景特效：**
- 无（默认）
- 金币雨

### 2.5 音效提醒

| 功能 | 说明 |
|------|------|
| 启用开关 | 可开启/关闭金币音效 |
| 触发金额 | ¥10 / ¥50 / ¥100 三档可选 |

**触发效果：**
- 播放金币音效（合成音效）
- 掉落金币动画

### 2.6 显示界面

**显示内容：**
- 当前时间（按秒更新）
- 当前日期
- 大金币图标
- 实时工资金额
- 每秒收入速率
- 工作时长统计
- 计算模式标识

**交互：**
- 实时预览（设置面板右侧）
- 完成设置（全屏显示）
- 屏保模式（全屏 + ESC退出）
- 支持横屏/竖屏

## 3. 技术规格

**技术栈：**
- HTML5 + CSS3 + JavaScript
- PWA（Service Worker）
- Web Audio API（合成音效）

**兼容平台：**
- iOS Safari（添加到主屏幕）
- Android Chrome
- 桌面浏览器

**文件结构：**


## 5. 使用说明

### 手机端使用
1. 用浏览器打开 `index.html`
2. 点击「添加到主屏幕」
3. 点击桌面图标打开

### 桌面端使用
1. 直接用浏览器打开 `index.html`
2. 点击「完成设置」或「进入屏保模式」
3. 按 ESC 退出全屏
