
# 实时工资 - 产品需求文档（PRD）

## 1. 产品简介

实时工资是一款可视化展示工资收入的网页应用，支持多种主题、背景特效和音效，适合桌面和移动端。

---

## 2. 功能需求

### 2.1 基础设置

| 功能 | 说明 |
|------|------|
| 月薪输入 | 支持输入工资金额 |
| 货币单位 | 支持人民币(¥)和美元($) |
| 时区选择 | 支持北京、伦敦、纽约、东京、洛杉矶 |

### 2.2 收入计算方式

| 模式 | 说明 | 计算规则 |
|------|------|----------|
| 24小时计算 | 全天每秒递增 | 月薪 ÷ 当月天数 ÷ 86400 |
| 工作时间计算 | 仅在工作日和上班时间递增，午休不计 | 仅在设置的工作时间内递增 |

### 2.3 工作时间设置

| 设置项 | 默认值 |
|--------|--------|
| 上班时间 | 09:30 |
| 下班时间 | 19:00 |
| 午休开始 | 12:30 |
| 午休结束 | 14:00 |

### 2.4 主题与视觉

- 支持暗黑、亮色、金色、绿色、紫色主题
- 响应式设计，适配桌面和移动端

#### 背景特效

- 金币图案雨（默认，SVG金币）
- 金币符号雨（¥）
- 美元符号雨（$，金色）
- 星光 ✨
- 光芒 🌟
- 气泡 🔮
- 背景特效可一键开关

#### 金币音效

- 金币音效可一键开关
- 支持触发金额设置（¥10/¥50/¥100/¥1000）

### 2.5 显示界面

- 当前时间、日期
- 大金币图标
- 实时工资金额
- 每秒收入速率
- 工作时长统计
- 计算模式标识

### 2.6 交互

- 实时预览（设置面板+右侧预览）
- 完成设置后全屏展示
- 屏保模式（全屏+ESC退出）

### 2.7 访问与部署

- 访问地址：[www.mysalary.site](https://www.mysalary.site)
- 支持 GitHub Pages/Vercel/Netlify 等静态部署

### 2.6 显示界面

**显示内容：**
- 当前时间（按秒更新）
- 当前日期
- 大金币图标
- 实时工资金额
- 每秒收入速率
- 工作时长统计
- 计算模式标识

**交互：**
- 实时预览（设置面板右侧）
- 完成设置（全屏显示）
- 屏保模式（全屏 + ESC退出）
- 支持横屏/竖屏

## 3. 技术规格

**技术栈：**
- HTML5 + CSS3 + JavaScript
- PWA（Service Worker）
- Web Audio API（合成音效）

**兼容平台：**
- iOS Safari（添加到主屏幕）
- Android Chrome
- 桌面浏览器

**文件结构：**
```
money/
├── index.html      # 主应用
├── manifest.json  # PWA配置
├── sw.js         # Service Worker
├── icon.svg      # 应用图标
└── PRD.md       # 产品需求文档
```

## 5. 使用说明

### 手机端使用
1. 用浏览器打开 `index.html`
2. 点击「添加到主屏幕」
3. 点击桌面图标打开

### 桌面端使用
1. 直接用浏览器打开 `index.html`
2. 点击「完成设置」或「进入屏保模式」
3. 按 ESC 退出全屏
