# 实时工资（Real-time Salary）

一个展示实时工资收入的网页应用，支持多种背景特效和音效，适合桌面和移动端。

## 访问地址

👉 [www.mysalary.site](https://www.mysalary.site)

## 功能特性

- 实时计算并展示工资收入
- 支持 24 小时/工作时间两种模式
- 支持多币种（人民币、美元）
- 多种背景特效（金币雨、美元雨、星光、光芒、气泡、金币图案）
- 背景特效可一键开关
- 金币音效可一键开关，支持触发金额设置
- 屏保模式/全屏模式
- 暗黑/亮色/多主题切换
- 响应式设计，适配桌面和移动端

## 使用说明

1. 打开 [www.mysalary.site](https://www.mysalary.site)
2. 设置月薪、币种、工作时间等参数
3. 可切换背景特效和音效
4. 支持全屏/屏保展示

## 部署

本项目为纯前端静态页面，支持 GitHub Pages、Vercel、Netlify 等平台一键部署。
